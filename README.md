# chefbot
Szakdolgozat

A program PHP, mysql és bootstrap használatával készült.

A webes alkalmazás egy receptkönyvet valósít meg, ahol a regisztrált felhasználók tölthetik fel a recepteiket és értékelhetik másokét. A projekthez egy összetetteb adatbázis tartozik, amiben előre definiált hozzávalókból pickelhet recept írásakor a user és így számolja ki a rendszer egyes ételek energia,kalória..ect tartalmát. Későbbiekben szeretnék olyan ajánlást is beletenni aweboldalba, ami megnézné a felhasználó kedvenc recepteit és azokban összefüggést keres (pl: mindegyikben van répa, krumpli..stb) és olyan, addig általa nem látott receptet ajánl fel amiről feltételezhetjük, hogy szeretni fogja.
